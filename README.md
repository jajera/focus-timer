# Focus Timer

Day 4 of Kiro Birthday Week: Build one app with one sentence.

## Spec Prompt

> Build a simple static focus timer web app with a large countdown, a minutes input defaulting to 25, start pause and reset controls, and clear Idle Running and Paused states that lock the duration while active.

## Run Locally

Open `index.html` in any browser — no build step or server required.

## Run Tests

```bash
npm install
npm test
```
